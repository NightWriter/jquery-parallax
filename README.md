jQuery-parallax
=============

jQuery-parallax is a plugin for easy parallax effect creation.

Using:
-------

	var inmain = new jQuery.parallax({                                            subLayers: [                                                        [".paralax-layers-1",0.02],                                                        [".paralax-layers-2",0.06],                                                        [".paralax-layers-3",0.2]                                                        ]                                        });        inmain.init();
